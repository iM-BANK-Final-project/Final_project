# Security Policy

## Sensitive Data

이 프로젝트는 법인고객 데이터를 다룰 수 있으므로 다음 정보는 저장소에 커밋하지 않습니다.

- 원본 고객 데이터
- 고객명, 사업자번호, 연락처, 주소 등 직접 식별자
- 계약번호, 계좌, 내부 고객 ID 등 재식별 가능한 키
- API key, DB 접속 정보, 토큰, 비밀번호
- 내부 시스템 URL 또는 접근 권한 정보

## Reporting

보안 이슈나 민감정보 노출 가능성을 발견하면 공개 이슈에 상세 데이터를 올리지 말고 팀 내부 채널로 먼저 공유합니다.
