# iM Bank 업무 페이지 배경 디자인

## 목표

Overview, 지속거래약화 예측, CRM 우선순위, 맞춤추천, AI 리포트의 다섯 업무 페이지 배경을 동일한 iM Bank 브랜드 계열로 통일한다. 색감은 분명히 보이되 카드와 표보다 강하지 않은 중간 농도를 사용한다.

## 시각 규칙

- 업무 화면의 기본 배경은 `#F5FBF9`에 가까운 밝은 민트 화이트다.
- 좌측 상단에는 iM Bank 민트 `#00C7A9`를 약 12~13% 농도로 배치한다.
- 우측 하단에는 로고의 연두 계열을 약 9~10% 농도로 배치한다.
- 배경은 viewport 기준으로 고정하여 페이지 콘텐츠 높이가 달라도 같은 색감과 위치로 보이게 한다.
- 카드, 표, 내비게이션의 기존 반투명 흰색과 가독성은 유지한다.
- 로그인 페이지의 별도 흰색·브랜드 배경은 변경하지 않는다.

## 구현 방식

- `styles.css`에 업무 화면 전용 배경 디자인 토큰을 추가한다.
- `body`의 기본 배경은 밝은 단색으로 두고, 인증 후 `.app-shell`에 고정된 다중 그라데이션을 적용한다.
- 배경 외의 페이지별 컴포넌트와 데이터 흐름은 변경하지 않는다.

## 검증

- CSS 테스트로 `.app-shell`이 iM Bank 민트·연두 배경 토큰과 `background-attachment: fixed`를 사용하는지 확인한다.
- 로그인 화면에 기존 독립 배경 규칙이 남아 있는지 확인한다.
- 전체 프론트 테스트와 프로덕션 빌드를 실행한다.
